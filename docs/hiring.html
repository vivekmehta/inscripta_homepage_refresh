AI Engineer (NLP)

We are looking for an NLP Engineer to work with us on cutting-edge problems in language processing. The ideal candidate should be:

  • Capable of building end-to-end, deployable NLP solutions, preferably in Python (should be able to adapt to other programming languages as needed)
  • Familiar with standard NLP toolkits (Spacy, NLTK, etc.), pipelines (morphological analysis, POS tagging, parsing), and data curation
  • Enthusiastic about analyzing text data and understanding linguistic features specific to the domain
  • Fluent with modern deep learning architectures (CNNs, LSTMs, Transformers etc.) and Tensorflow
  • Capable of adapting and implementing techniques from the research literature
  • (Optional) Conversant with cloud frameworks such as GCP, Azure and Sagemaker

  • Gain exposure to delivering NLP-centric real-world solutions
